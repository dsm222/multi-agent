# project_scope

## 研究对象
- LLM 驱动的多智能体协作推理系统（Multi-Agent ReAct）。

## 贡献边界（冻结）
- 仅做编排 / 通信 / 验证 / 调试层创新。
- 不做底模训练与大规模预训练改造。

## 三组件主线
1. 动态拓扑路由
- 每回合围绕 round goal 生成通信子图，仅沿激活边通信，降低无效广播。

2. vote-first + verifier
- 默认多数投票汇总候选答案。
- verifier/inspector 做一致性与证据检查，仅在冲突时触发短辩论纠错。

3. trace + replay
- 记录结构化轨迹日志（state/messages/router/vote/verifier）。
- 支持失败任务重放与中间节点干预复现实验。

## 不纳入当前阶段主线
- GNN/RL 路由训练作为后续拔高线，不纳入阶段0~2交付。

## 阶段0验收口径（30秒）
- 本课题不做底模训练，聚焦多智能体编排系统；主创新是动态通信拓扑、验证型共识和可重放调试。
